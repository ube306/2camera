## 2Camera 到底是什么？

说白了，就是一个让你 **同时调用手机前后摄像头** 进行拍照或录像的相机 App。

手机默认相机只能同时用一个镜头，切换来切换去麻烦不说，很多「此时此刻」的双重感受也就这么丢了。2Camera 不一样，它把两个镜头的画面同时搬上屏幕，前摄、后摄各司其职，一次按下快门，三张照片（前置 + 后置 + 合成）统统存好。

开发者是国内独立开发者，App 在 iOS 平台上持续迭代，目前已完整支持 iOS 17 / 18 / 26，最新版本还适配了苹果 iOS 26 的液态玻璃（Liquid Glass）设计风格，用起来感觉跟系统原生一样顺滑。

支持设备覆盖面也挺广，iPhone 11 起、一直到最新的 iPhone 17 系列，还有 iPad Pro / Air / mini，基本上家里的老设备也能用上。

## 功能不只是「两个镜头同时开」

第一次听说这个 App，很多人觉得：不就是画中画？其实细节挺多的。

**多种分屏布局**，你可以选画中画（PiP 小窗，可以自由移动位置、调整大小，圆形/矩形/正方形都支持）、上下分屏、左右分屏，甚至单摄像头模式。不同场景选不同布局，Vlog 博主、亲子记录、直播讲解都能找到合适的。

**一键拍 3 张**，按一下快门，同时保存：前置单独一张、后置单独一张、合成双画面一张。以后整理相册的时候三份都有，不用担心漏。

**可调拍摄参数**，支持 4K 超高清录像、1080P 拍摄、调节帧率、运动防抖、音频声道质量这些对视频质量有影响的设置。还能固定调用主摄/超广角/长焦镜头，适配不同场景。

**自定义水印**，支持添加时间、地理位置、坐标、海拔这些水印内容，位置透明度都能改。喜欢在照片上留下「时光印记」的人会挺喜欢。

**快速启动**，锁屏小组件、桌面小组件、控制中心、相机控制键都能直接唤起，不用每次从 App 列表里翻。
